# 今回演習として作成するシステムの要件
- 為替のレポート機能に、多国籍通貨に対応できるように機能を追加する。

具体例は以下  
元々の機能としては以下を出力する機能。

| 銘柄 |  株数  |  価格  |  合計  |
|----| ---- | ---- | ---- |
| IBM |  1000  |  25  |  25000  |
| GE |  400  |  100  |  40000  |
|    |    |  総計  |  65000  |

上記を機能追加し、以下のように出力するようにする。

| 銘柄 | 株数   | 価格      | 合計         |
|----|------|---------|------------|
| IBM | 1000 | 25 USD  | 25000 USD  |
| GE | 400  | 150 CHF | 60000 CHF  |
|    |      | 総計      | 65000 USD  |

上記表示を実現するために、為替レートデータの作成も必要となる。

| 換算元 | 換算先 | レート |
|-----|-----|-----|
| CHF | USD | 1.5 |
